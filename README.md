## 📚 О курсе

Этот репозиторий — не просто roadmap, а **полноценный курс по Go из 13 модулей**, разбитый по урокам в папке [`/course`](./course).

Каждый модуль содержит:

* 🎯 **Цель** — что ты будешь уметь после модуля
* 📚 **Теорию** — что и где читать/смотреть
* 💻 **Практику** — конкретные задания и проекты
* ✅ **Чек-пойнт** — самопроверка перед переходом дальше
* ⚠️ **Красные флаги** — типичные ошибки, которые тормозят рост

Курс ведёт от установки Go до уверенного прохождения Senior-собеседований за 10–14 месяцев при 2–3 часах в день.

👉 **Старт:** [Модуль 0 — Фундамент](./course/00-fundamentals.md)

## 🧭 Как проходить курс

* Открой [Модуль 0](./course/00-fundamentals.md) и пройди его до конца, отмечая чекбоксы внутри.
* Для каждого модуля доводи проект до конца — незаконченный проект учит хуже законченного «плохого».
* После каждого проекта сверяйся с чек-пойнтом в конце модуля.
* Пройдя модули 0–7, иди на первые Junior-собеседования — реальный фидбэк ускоряет рост.
* Не стесняйся возвращаться к ранним модулям: глубина приходит со вторым проходом.

## ⏱️ Длительность

| Грейд | Срок (2–3 ч/день) | Что умеешь |
|------|--------------------|-----------|
| 🟢 Junior | 4–6 месяцев | CRUD API, базовая работа с БД, простые горутины |
| 🟡 Middle | 8–12 месяцев | Архитектура сервиса, контекст, тесты, профилирование |
| 🔴 Senior | 2+ года практики | Дизайн систем, наставничество, глубокий runtime |

> Время — ориентир. Один проходит за 6 месяцев, другой за 14. Норм оба варианта.

---

## 📌 3 железных правила

* **Не прыгай вперёд.** Go обманчиво простой — без фундамента упрёшься в потолок на собесах.
* **Пиши код сам.** Подсмотреть можно только после часа борьбы с задачей.
* **Делай проекты до конца.** Незаконченный проект учит хуже законченного «плохого».
